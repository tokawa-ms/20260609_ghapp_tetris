# 20260609_ghapp_tetris

## ここまでに実装した内容

このリポジトリは、HTML ファイルをそのまま開くだけで遊べるシンプルな Tetris Clone です。

### 主要な実装内容
- 10x20 のゲーム盤とテトリミノの落下・回転・移動
- 左右移動 / 回転 / 高速落下 / 直下落下
- スコア、消したライン数、レベル表示
- 次ブロックプレビュー
- 一時停止・リセット機能
- 追加ライブラリ不要の単一ファイル構成

### 実行方法
1. `index.html` をダブルクリックで開く
2. もしくは `python -m http.server 8000` で起動し、`http://127.0.0.1:8000/index.html` を開く

### MAI-Code-1-Flash でやったこと
MAI-Code-1-Flash では、以下の内容を中心に実装と整理を行いました。
